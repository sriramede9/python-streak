Level 1:
groupby one column
→ summarize

country → average points


Level 2:
groupby multiple columns
→ drill into dimensions

country + province → average points


Level 3:
groupby + agg
→ build reports

country + province → score + price + count


Level 4:
groupby + transform/filter
→ create insights

"is this wine above its country's average?"

A good rule:
After sorting/filtering and you care about the first N rows → use iloc
After sorting/filtering and you want to preserve the original index meaning → use loc
If the old index is now useless → reset_index(drop=True)
